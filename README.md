# 사이버보안과 Git 실습

이 저장소는 사이버보안과 학생들을 위한 Git 실습 자료를 제공합니다.

## 사용 방법
1. 이 저장소를 자신의 계정으로 포크(Fork)하세요
2. 포크한 저장소를 로컬에 클론(Clone)하세요
3. 각 실습 지시사항에 따라 과제를 수행하세요
4. 변경사항을 커밋하고 푸시하세요
5. 제출 기한까지 풀 리퀘스트(Pull Request)를 생성하세요
